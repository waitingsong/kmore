Interface Kmore<D, Dict>

Generate knex method refer to tables. method name from keyof T, ReturnType is the type according to the key, props both enumerable and unenumerable.

description

T = { user: {id: number, name: string} } will get db.user() => Knex.QueryBuilder<{id: number, name: string}>
